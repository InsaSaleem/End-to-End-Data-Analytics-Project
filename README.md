# 🚀 Project Breakdown

## 🧩 Task 1 — Automated Web Scraping

### Objective

Develop an automated web scraping pipeline to collect large-scale book catalog data from online web pages.

### Implementation

* Performed HTTP requests using Requests
* Parsed HTML structures using BeautifulSoup
* Implemented dynamic pagination handling
* Automated multi-page data extraction

### Output

Successfully extracted and stored:

* Book Titles
* Prices
* Availability Status

Dataset Size:

* 1000+ records exported into CSV format

---

## 🔍 Task 2 — Exploratory Data Analysis (EDA)

### Objective

Perform statistical analysis and data quality assessment on the Titanic dataset.

### Implementation

* Data cleaning and preprocessing
* Missing value analysis
* Median-based imputation techniques
* Correlation and survival analysis

### Key Insights

* Passenger class significantly influenced survival rates
* Female passengers showed higher survival probability
* Missing data patterns were successfully identified and handled

---

## 🎨 Task 3 — Data Visualization & Dashboarding

---

## 🧠 Task 4 — Sentiment Analysis & NLP

### Objective

Build a Natural Language Processing framework to analyze customer sentiments from textual reviews.

### Implementation

* Text preprocessing and cleaning
* Regex-based token normalization
* Sentiment scoring using:

  * VADER
  * TextBlob

### Output

Generated:

* Sentiment Distribution Dashboard
* WordCloud Visualizations
* Polarity vs Subjectivity Analysis

### Performance Highlight

Achieved approximately **92% agreement accuracy** between VADER and TextBlob sentiment predictions.
